-비동기 실행
동기(Synchronous)
  프로그램에서 명령어가 차례대로 실행
	class A{
		1.void a()
		2.void b()
		}

비동기(Asynchronous)
	프로그램에서 명령어가 순서에 관계없이 실행되거나 
	다른 프로그램이 작동할때 background에서 실행중인 상태
	-->이런 비동기 상태로 만드는 클래스가 Thread
	--> 주 사용 예는 GUI가 있는 프로그램(swing, java FX)는 만들 때 어떤 이벤트가 오기를 대기할 때
	     어떤 이벤트가 오기를 기다릴때

이벤트, 이벤트 핸들링, 이벤트 리스너
예) 버튼을 누른다, 마우스로 클릭, 드래그를 한다, 키보드를 누른다 등
이벤트를 컨트롤 하는게 이벤트 핸들링-->이벤트 핸들링을 하는 프로그램을 보통 이벤트 리스너라고 한다
이벤트 리스너는 이벤트가 일어날때까지 대기(wait)-처리
